Global Variables

  • Global Variables are visible everywhere
  • If you use an undeclared variable inside a function it automatically becomes Global
  • Global Vars automatically become window vars ie window.var1 and .var1 are the same

Local Variables

  • Variables declared inside a funtion cannot be accessed globally
  • Once JS exits out of the function, the variable is trashed

Events

  • Happen inside a script and / or HTML pages
  • JS reacts to events such as onclick="", onmouseover="", onkeydown=""
  • Events can be linked to muliple elements for instance Click Me
