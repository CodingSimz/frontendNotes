String Properties

Length, returns total length of a string including: chars, spaces, special chars, escape sequences such as \" (here \" is one char), the number of chars in a html entity (here > will add 4 not 1 to the length), and number of chars used in html tags (for instance, <em> will add 10 to the length). This is important when using the length property or string methods on HTML code.

String Methods

indexOf() finds a string within a string, here count starts at 0 left-to-right

Please find the word 'Find' in this text. note this locates Find, find would return 7. Also, count includes html markup not visible to the User

lastIndexOf() searches backwards for string within string

replace("old", "new") this is case sensitive, to override use regular expression with /i flag i.e. no string "" just /OLD/i and this method ONLY replaces the first match

search() is quite similar to indexOf but its has much greater flexibility when working with Regular Expressions.

slice(start, end) if paremeters are -tive it will count from the right, if only one number is given slice(num) will behave like indexOf or lastIndexOf when -ve, and you could use a function to place a variable as a paramenter rather than a number e.g.
function findStr2() {
  var ingreds = "chocolate vanillia ginger";
  var spacepos = ingreds.indexOf(" ");
  var flavour = ingreds.slice(0,spacepos);
  document.getElementById("an02").innerHTML = flavour;
  } will end at first space in the list "chocolate vanillia ginger"
