Arrays

In JS, arrays use numbered indexes accessed by [n]. Arrays can be initialised simply by var array_name = [item1, item2, ...]; and can include mixed data types eg var person = ["John", "Doe", 46];.

  • Full array
  • Specific element
  • An element can be changed via index number [n] eg fruits[2] = "Kiwi";

Array Length

  • arr_name.length number of items in Array
  • var last = arr_name[arr_name.length - 1]; will access the last element in the array
  • arr_name[arr_name.length] = item; will add a new element to the array

Array Methods include

  • sort() returns alpha
  • valueOf() and toString() returns array as comma separated str.
  • join() returns a string with a separator of choice e.g. classes.join(" then ") returns
  • shift() removes first element and reindexes
  • unshift() adds first element and reindexes
  • pop() removes last element eg fruits.pop()
  • push(element) appends an element eg fruits.push("Kiwi")
  • reverse() displays in reverse order
  • splice(p1,p2) where p1 is the starting point, and p2 number of elements to remove from the array; this is a clean removal (rather than using delete keyword)
  • splice(p1,p2,pn,...) where p1 position in array to splice in, p2 how may elements to remove, pn defines new elements to be added e.g. fruits.splice(2, 1, "Lemon", "Kiwi") will delete 1 item in position 2 and add Lemon and Kiwi. Note if you store this in a variable you can access the removed item eg var remove = fruits.splice(2, 1, "Lemon", "Kiwi"); removed the [2] fruit (apple)
  • concat() will merge existing arrays into a new array eg var myChildren = myGirls.concat(myBoys); and can take muliple array arguments var myChildren = arr1.concat(arr2, arr3);
  • Beware using sort() method on number arrays eg
    var nums = [40,100,1,5,25,400,10]; nums.sort(); instead use a Compare fn such as nums.sort(function(a,b){return a-b}); which will return This compare fn will compare all the values in an array two values at a time and return a -ve, 0 or +ve value depending on the arguments and sort lowest to highest

You can use the for loop to iterate through an array

Objects

In JS Objects use Named Indexes. Objects use element:member pairs stored in {} for instance:
var person = {firstName:"John", lastName:"Doe", age:46};
