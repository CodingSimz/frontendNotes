Switch Statements

Syntax Ex var age = 50; var greeting ="";
switch(expression) {
  case x: //do or displ
   break;
  case y: //do or displ
   break;
  default: //do or displ
 } 		switch(age) {

   break;
  case 60: greeting = "Yay 60!"

  • The break keyword stops the switch block,s execution
  • The default keyword specifies what code to run/display if there is no case match, it can go anywhere or be absent from the switch statement
  • If multiple cases match the expression value, the first case is selected
  • If there is no default and no cases match, the pgm continues to the statements following the switch

The For Loop

The for loop iterates through a block of code, such as an array and has the following syntax
for (statement 1; statement 2; statement 3) { // code block to be executed }

  • Statement 1 (optional) is executed once, before the code block and so can be used to set variables (var keyword not required); mulitple variables shoud be separated by a comma
  • Statement 2 (optional) defines the condition for executing the code block. If statement 2 returns True the loop continues, if it returns False the loop will end. Although this statement is optional without it the codeblock must contain a break or else the loop won't end
  • Statement 3 (optional) is executed after each execution of the code block for instance, to increment i++ or decrement I-- the variable - this can be omitted here and placed in the loop.

The For Loop format is often
for (i = 0; i < 5; i++) { // code block to be executed }

break keyword: breaks out of a loop, for example, this loop contains an if statement with a break when i=3

continue keyword: jumps past an iteration in a loop, for example, this loop contains an if statement with a continue when i=3

Labels in JS

A label is simply an identifier followed by a colon (:) that is applied to a statement or a block of code. Syntax labelname: {statement list}. A use case is combining labels with break and continue keyworkds in nested loops
outerloop: //label name
for() {
 innerloop: //label name
 for() {
  if () break;
  if () break innerloop; //quit inner
