JavaScript will dynamically convert numbers to a string. This is called implicit type coercion and is triggered when + is used and one of the operands is a string, but this isn't always a good idea.

To control typcasting use n.toString() method, e.g. "text " + (100 + 10).toString() + " ." or the String(n) object.

Exponential notation: very large or small numbers can be written in exponential notation. var a = 123e5; and var b = 123e-5; will display as , similarly n.toExponential(n) returns the number rounded off and presented with scientific notation var a = 9.656; will return using n.toExponential(2)

n.toFixed(n) returns a str with n length after decimal point whereas n.toPrecision(n) returns a str with specified number of significant digits.

Math.PI.toFixed(2); vs Math.PI.toPrecision(2);

Converting Strings to Numbers

Use parseInt() to convert str to WHOLE numbers for instance parseInt("16.22 10 years"); // returns 16 Note: Only the first number is returned, to parse decimals use parseFloat() ie floating point numbers. In both cases parsing ("years 16.22 10"); // returns NaN because the string began with text.

The valueOf() method converts number objects to primative vals but is rarely used in code.

Basic Maths

Math.random() returns ie random number between 0 - 1

Math.min(9,6,4,13,2,16) returns lowest n

Math.max(9,6,4,13,2,16) returns highest n

Other math methods include Math.round(n) Math.ceil(n) Math.floor(n) Math.abs(n) Math.pow(x, y) Math.sqrt(n)
