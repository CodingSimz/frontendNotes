CSS Selectors

When we write multiple selectors:

  • div, p comma separates elements, used for applying properties to multiple element tags at once
  • div p without a comma to separate each is a descendant or child of the one before eg will control all p elements with that div. DESCENDANT SELECTOR
  • div + p with a + sign will refer to the first element after eg first paragraph after the div. ADJACENT SIBLING COMBINATOR
  • div > p with a > sign will refer to all p elements the div is a parent of but will only look level down. CHILD COMBINATOR SELECTOR
  • p ~ ul with a ~ sign will refer to all ul elements that are preceded by a p element. THIS IS AN GENERAL SIBLING
