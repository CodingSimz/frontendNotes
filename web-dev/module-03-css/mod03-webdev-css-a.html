CSS basics

CSS (cascading style sheets where declaration syntax is selector and name:value pairs separated by semi-colons.

Selector {property: value; }
  • # specifies id and should only be applied to one html element
  • . specifies class and multiple classes can apply to multiple html elements

There are 3 ways to apply stylesheets inline using the style tag, internal using the style element in the document header, and external using a .css file that is linked to the documents in the header.

Link States using <a> tag

There are style order rules for link states:
a:hover MUST come after a:link and a:visited
Also, a:active MUST a:hover.
In other words set link, visited, hover, active or the browser may not show state changes in response to User activity.

In the a tag you can set a target attribute to control how links open

target="_blank" <!-- opens link in a new window -->
target="_self" <!-- opens link in actual window -->
target="_parent"
target="_top" <!-- both handle frames -->

The CSS Box Model

This model treats each element as a collection of 4 concentric boxes: Margin, Border, Padding, Content. The Corresponding properties are height, width and style.

This is a div with box model applied.
Note if you do not specify border style you cannot apply a width.

NOTE: a) Border shorthand is border: width style color; b) Border-radius must be applied separately, and c) there is an outline property that can be applied OUTSIDE the border (show in double red lines).
