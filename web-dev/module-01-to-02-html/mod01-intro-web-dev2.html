The DOM provides programatic API for controlling a browser and accessing the contents of a web page: this includes:

  • Finding and setting values of elements on a page
  • handling event controls
  • modifying styles assoc'd with elements
  • serializing and deserialising a page as an XML document
  • Validating and updating web pages

To modify a page element

  1. Create a new object containing the new data
  2. find the parent element that should contain the new data
  3. append, prepend, insert or replace the data

To remove an element or attribute

  1. Find the parent element
  2. Use removeChild or removeAttribute
