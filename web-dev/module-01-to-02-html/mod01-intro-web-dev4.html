Software Development

Software Development Strategies can be applied to any type of development process or project

Software Development Life Cyle - used to be simpler because the tech and Users had simpler demands but today its more complex so Design is more important than ever!

3 Methods of the SDLC are RAD, Waterfall and Inerative.

The Key to Software Development is to have as much upfront information as possible.

Remeber different tasks and projects require different methodologies and resources - Software Development Life Cycle templates are often are great place to start.

RAD

Rapid Application Development - functional but not very detailed.

Take requirements, develop, QA and then deploy.

Waterfall

Gather requirements, analysis, design, development, QA, Deployment

Issue with this method is that it implies version one is perfect,

-^__^-

Interative

Ever-evolving = repeating the cycle for each version.

Also known as the Spiral Model.

  1. Initial Planning Phase - sometimes abstract ideas
    faster checkout
  2. Tech, money, time, resources involved
  3. Requirements Analysis: more detailed breakdown of above and determine of Best Method to reach goals
  4. Design and initial development
  5. Implementation: platform (web, mobile etc)
  6. Testing - both manual and automated - performance, usability, integrity
  7. Evaluation UAT User Acceptance Testing
  8. User Sign-off: did we deliver?
  9. Support Cycle - how will you maintain from this point forward?
