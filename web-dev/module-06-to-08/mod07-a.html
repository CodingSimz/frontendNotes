We have Combinators:

  1. Descendant .a b {} affecting all b nested in a
  2. Direct child .a >b {} affecting ONLY direct
  3. Adjacent sibling a + p {} affecting next p after a
  4. General sibling a ~ p {} affecting all ps after a

We have Pseudo-Elements:

selector::pseudo-element {property: value;}
  1. ::first-line for block lvl elements only
  2. ::first-letter for block lvl elements only
  3. ::before and ::after
  4. ::marker pseudo-element selects the markers of list item
  5. ::selection matches the portion of an element that is selected by a User
